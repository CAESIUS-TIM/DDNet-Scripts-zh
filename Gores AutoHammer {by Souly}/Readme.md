这是Souly的Gores AutoHammer版本。（使用钩子时切换到枪，开火时切换到锤子）

你也可以只使用 weapons.cfg，但那样你就无法使用绑定切换开/关。

要初始化它：执行 switch.cfg，然后在游戏中使用 `CTRL+G` 来切换开关。
