# 分身卡锤子

## 说明

启用/禁用强制分身使用锤子的切换。

## 使用方法

要启动它，请执行 `exec HammerTakeOff.cfg`，然后在游戏中使用6来切换开关。

## 原README

Enables/Disables Forcing switch to Hammer on Dummy toggle.

To initiate it `exec HammerTakeOff.cfg` , then use 6 in-game to toggle it on & off.
